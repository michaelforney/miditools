ALSASEQIO(1) General Commands Manual ALSASEQIO(1)

NAME

alsaseqio — ALSA sequencier I/O

SYNOPSIS

alsaseqio [-rws] [-n name] [-f rfd[,wfd]] [-p client[:port]] [command...]

DESCRIPTION

alsaseqio is a bridge from MIDI byte stream(s) to an ALSA sequencer port.

If command is specified, alsaseqio will set up pipes on rfd and/or wfd that can be used to read and/or write MIDI messages. Then, it will fork to the background and execute command in the foreground.

Otherwise, alsaseqio will read MIDI messages from the sequencer port, writing them to wfd, and read MIDI messages from rfd, writing them to the sequencer port.

OPTIONS

-r
Read from the ALSA sequencer port.
-w
Write to the ALSA sequencer port.
-s
Allow subscription to alsaseqio's port, even a target port was specified with -p.
-n
The ALSA sequencer client name and port name to use. Defaults to alsaseqio.
-f
The file descriptors on which to read and/or write MIDI 1.0 byte streams. If only one file descriptor is given, then it is used for both reading and writing. If not specified, defaults to standard input (0) for reading and standard output (1) for writing.

If command is specified, these file descriptors refer to the ends of the pipes on the command side. In -r mode, MIDI messages can be read from rfd, and in -w mode, MIDI messages can be written to wfd.

Otherwise, in -r mode, MIDI messages read from the sequencer port are written to wfd, and in -w mode, MIDI messages read from rfd are written to the sequencer port.

-p
The target ALSA sequencer port. The client can be specified by its number, its name, or a prefix of its name. The port is specified by its number, and can be omitted in which case 0 is used. If -p is not specified, subscription will be enabled for alsaseqio's port, allowing other clients to connect.

If neither -r nor -w is specified, the default is -rw.

EXAMPLES

Hex dump any MIDI messages received.

alsaseqio -r | od -t x1

Play a C note for one second on a synthesizer named ‘MODEL D’.

{ printf '\x90\x3C\x7F'; sleep 1; printf '\x80\x3C\x7F'; } | alsaseqio -wp 'MODEL D'

Start oscmix(1) using port 1 of device ‘Fireface UCX II’.

alsaseqio -p 'Fireface UCX II:1' -f 6,7 oscmix

Bridge an ALSA sequencer port to a CoreMIDI port on a networked Mac.

alsaseqio ssh mymac coremidiio
